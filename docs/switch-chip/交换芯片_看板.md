# 交换芯片行业看板

## 一、版块概览

### 1.1 产业链结构
```
SerDes IP (Synopsys/Cadence)
        ↓
交换芯片设计 (Broadcom/Marvell/Nvidia/Cisco/盛科)
        ↓
代工 (TSMC 5nm/3nm 为主)
        ↓
先进封装 (CoWoS-S/L, 部分 InFO)
        ↓
整机厂 (Arista/Cisco/Juniper/工业富联/锐捷/紫光新华三)
        ↓
云厂/运营商 (AWS/Azure/GCP/Meta/字节/阿里)
```

### 1.2 核心驱动力
- **AI Fabric 内卷**：Scale-out 跨柜以太网 + Scale-up 柜内（NVLink/UALink/Tomahawk-Ultra）
- **代际跳跃**：51.2T（2023-24 主流）→ 102.4T（2025-26 量产）→ 204.8T（2027+）
- **SerDes 升级**：112G PAM4（当前）→ 224G PAM4（2025-26）→ 448G LPO/CPO 路线分叉
- **光电协同**：CPO（共封装光学）让交换芯片 + 光模块在 substrate 上合体

## 三点五、厂商核心能力拆解：SerDes 是关键吗？

**核心论点**：SerDes 是"必要不充分"——是入场券和能力天花板，但不是护城河本身。真正的核心能力是几个"难的东西"叠在一起的系统级整合。

- **为什么必要**：200G/224G 这代 SerDes 决定能不能做到 102.4T、信号能不能干净打出去，做不出连牌桌都上不了。
- **为什么不充分**：Credo / Alphawave 有世界级 SerDes 却非交换龙头——只有 I/O，没有交换架构和软件生态，最后只能卖 IP / 做 chiplet。

### 核心能力拆解（按护城河深浅排序）

| 能力 | 是什么 | 护城河 | 谁强 |
|------|--------|--------|------|
| **① 软件/生态** | SDK、SONiC、网络 OS 集成、可编程(P4)、遥测 | 最深 | 博通碾压（换芯片要重写软件，客户极不愿动） |
| **⑤ 客户绑定** | design win 多年锁定，进集群难替换 | 深 | 在位者通吃 |
| **② 交换架构** | buffer 架构、调度、拥塞控制、自适应路由、无损网络 | 深 | 博通(TH+Jericho)、中兴微(OLink) |
| **③ SerDes** | 高速 I/O、信号完整性 | 中 | 博通、Marvell、Nvidia（可买 IP 追赶） |
| **④ 制程+封装+规模** | 拿 TSMC 3nm/CoWoS 产能、每代 R&D 烧几十亿 | 中 | 博通、Nvidia |

> 真正的胜负手是 **①软件生态 + ⑤客户绑定** —— 这才是博通守住 ~70% 份额的根本。SerDes 强只是让你"有资格被考虑"。

### 核心能力因商业模式而异

| 厂商/模式 | 核心能力命脉 |
|-----------|--------------|
| 博通/Marvell（商用芯片） | SDK 软件生态 + SerDes + 架构 + 规模 |
| Nvidia（搭 GPU 卖） | GPU 主导力 + NVLink 生态锁定，不靠单卖交换取胜 |
| 思科（搭系统卖） | 整机系统 + 企业渠道 + 软件 |
| 盛科/中兴微（国产替代） | 国产替代卡位 + 客户深绑 + 追赶速度（窗口期"能量产+进供应链"比"性能对标"更重要） |

### 三层投资判断框架

看一家交换芯片厂商，别只盯 SerDes 速率（那只是性能参数），要问三层：
1. **技术够不够格**（SerDes + 架构）—— 入场券
2. **软件生态/客户有没有绑死** —— 真护城河
3. **商业模式决定它靠什么赚钱**（单卖芯片 / 搭 GPU / 搭系统 / 国产替代）

## 四、竞争格局

### Tab 1：Broadcom（统治者）
- 产品：Tomahawk 6（102.4T，2025/6 量产，含 CPO 版 Davisson）/ Tomahawk 5（51.2T 出货主力）、Jericho3-AI（深缓存，长距）、Tomahawk-Ultra（Scale-up）、Trident（园区）
- FY26Q2 指引：AI 半导体营收 ~$107亿（+140% YoY），AI 网络约占 40% ≈ $43亿/季；全季总营收 ~$220亿
- 2027 展望：AI 芯片营收破 $1000亿（注：2026 AI 营收估值因 Anthropic TPU 排除 rack 层，自 $62.5B 下修至 $55B）
- 看点：TH6 放量斜率、AWS/Anthropic 定制 ASIC 节奏、UALink 进展

### Tab 2：Marvell（追赶者 #1）
- 产品：Teralynx 10（51.2T，sub-600ns 延迟/radix 512/SONiC），Prestera（企业）；102.4T+1.6T 筹备中
- 份额：数据中心交换芯片 ~10-15%（Broadcom 之外主要替代者）；51.2T 年出货量预计冲向 180 万颗（650 Group）
- 看点：多供应商分散化采购红利，102.4T 产品落地节奏

### Tab 3：Nvidia（抢份额者）
- 产品：Spectrum-X（800G 以太主流）+ Spectrum-XGS（跨域）+ Spectrum-X Photonics（CPO）+ Quantum-X（InfiniBand）+ NVLink Switch（Scale-up）
- 战略：GPU+网络捆绑销售，凭借 GPU 主导力反向渗透网络
- 实际：网络营收 ~$50亿/季（FY26Q1，环比 +64%、同比 +56%），年化 run-rate 接近 $200亿

### Tab 4：Cisco（守成者）
- 产品：Silicon One G200（51.2T），自研芯片
- 看点：从依赖外购转向自研，但 AI 数据中心份额持续被 Broadcom/Nvidia 蚕食

### Tab 5：盛科通信 `688702`（国产以太网交换龙头）
- 产品：**Arctic 25.6T 已量产推广**（国产唯一可量产 25.6T，对标博通 TH4）；51.2T 在研，2026/5-6 月回片，2026H2 小批量；TsingMa.MX(2.4T)/GoldenGate(1.2T) 成熟创收
- 大单：阿里 3 万颗 25.6T 订单约 9 亿元（~3 万元/颗）；2026 三大云厂 25.6T 总需求 ~8 万颗，盛科独供
- 财务：2026Q1 营收 2.48亿元（+11.4%），预付款 3.96亿（环比+82%，订单先行指标）；2025 全年营收 11.51亿元（+6.35% YoY），归母净利 -1.50亿元，研发费用 6.79亿元（占营收 ~59%）
- 风险：代际落后全球（盛科 25.6T 量产 vs 全球 TH6 102.4T，约 1-2 代）；高强度研发阶段性压制盈利；51.2T 能否如期量产是关键变量
- *注：国产厂商全景详见正文「🇨🇳 国产交换芯片」模块*
